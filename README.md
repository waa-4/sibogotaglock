# Sibo Got a Glock v1

A small original fake-3D / raycaster browser shooter for GitHub Pages.

No Doom assets are used. It uses the uploaded Sibo/rust/zombie images in the `assets` folder.

## Controls

- WASD / arrows: move
- Q / E or arrow left/right: turn
- Mouse movement: turn after clicking the game view
- Click: shoot
- M: hidden mod menu
- R: restart after death/win

## Mobile

Basic touch buttons are included:
- turn left
- forward
- turn right
- shoot

## Map keys in `main.js`

```text
# = wall
. = empty
P = player spawn
Z = zombie
A = ammo refill
G = exit goal
```
